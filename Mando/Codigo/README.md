# Mando 春 Bot

## Ejemplos de código:

- **controlRobot**: Código para controlar el robot vía radiocontrol
- **buzzer**: Suena una melodía por el zumbador
- **encoder**: Imprime por la pantalla OLED el valor del encoder y su pulsador integrado
- **interruptoresLaterales**: Muestra los valores de los interruptores laterales
- **joystick**: Imprime por la pantalla OLED los valores de los joysticks ( eje Y ) y sus pulsadores integrados
