# Mando 春 Bot

## Ejemplos de código:
- __controlRobot__: Código para controlar el robot vía radiocontrol
- __buzzer__: Suena una melodía por el zumbador
- __encoder__: Imprime por la pantalla OLED el valor del encoder y su pulsador integrado
- __interruptoresLaterales__: Muestra los valores de los interruptores laterales
- __joystick__: Imprime por la pantalla OLED los valores de los joysticks ( eje Y ) y sus pulsadores integrados
