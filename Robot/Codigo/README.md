Ejemplos de código:

- **controlRobot**: Código para controlar el robot vía radiocontrol.
- **faros**: Enciende y apaga las luces al pulsar el botón central.
- **display7segmentos**: Incrementa o decrementa el número en función del pulsador derecho o izquierdo.
- **buzzer**: Toca una melodía distinta en función de si se pulsa el botón up, center o down.
- **pulsadores**: muestra el estado de los pulsadores por la pantalla oled.
- **temperatura**: Pinta la temperatura ambiente por el display 16x2. El sensor es el TMP36, aunque aceptaría LM35.
- **ultrasonidos**: Te da la distancia en cm leída del sensor de ultrasonidos por el display 16x2.
- **mandoTV**: Devuelve el código leído del mando de TV por el display 16x2. El número 1 enciende la luz del display, el 0 la apaga.
- **seguidorLineas**: El robot sigue una línea negra sobre fondo blanco ( grosor máximo de la línea, 15mm ).
