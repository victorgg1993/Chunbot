Ejemplos de código:
- __controlRobot__: Código para controlar el robot vía radiocontrol.
- __faros__: Enciende y apaga las luces al pulsar el botón central.
- __display7segmentos__: Incrementa o decrementa el número en función del pulsador derecho o izquierdo.
- __buzzer__: Toca una melodía distinta en función de si se pulsa el botón up, center o down.
- __pulsadores__: muestra el estado de los pulsadores por la pantalla oled.
- __temperatura__: Pinta la temperatura ambiente por el display 16x2. El sensor es el TMP36, aunque aceptaría LM35.
- __ultrasonidos__: Te da la distancia en cm leída del sensor de ultrasonidos por el display 16x2.
- __mandoTV__: Devuelve el código leído del mando de TV por el display 16x2. El número 1 enciende la luz del display, el 0 la apaga.
- __seguidorLineas__: El robot sigue una línea negra sobre fondo blanco ( grosor máximo de la línea, 15mm ).
